# QuantCore

A professional event-driven backtesting framework for quantitative trading research.

---

## Features

- Event-driven architecture
- Portfolio management
- Order execution
- Commission & slippage modelling
- Risk management
- Performance analytics
- Interactive Streamlit dashboard

---

## Tech Stack

- Python
- Pandas
- NumPy
- Plotly
- Streamlit

---
